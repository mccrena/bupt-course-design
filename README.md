# bupt-course-design
程序设计大作业---人脸识别检测器，可实现识别视频中人物的性别、年龄，以及人流总数的功能
# How to use
1.打开终端，输入并运行以下命令
```python
pip install -r requirements.txt
```
2.运行下述命令即可
```bash
python count.py
```
